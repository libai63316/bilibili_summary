## 4. timestamp 解析错误

### 问题描述
Paraformer 模型返回的 timestamp 格式与代码预期不符，报错 "'list' object has no attribute 'get'"。

### 原因分析
Paraformer 返回的 timestamp 是 list 格式 `[start, end, text]`，而代码期望的是 dict 格式。

### 修复代码
```python
if isinstance(ts, dict):
    start = ts.get('start', 0) / 1000
    end = ts.get('end', 0) / 1000
    segment_text = ts.get('text', '')
elif isinstance(ts, (list, tuple)) and len(ts) >= 3:
    start = float(ts[0]) / 1000
    end = float(ts[1]) / 1000
    segment_text = str(ts[2])
```

```python
    if _sensevoice_model is None or _sensevoice_model_device != device:
        _sensevoice_model = AutoModel(...)
        _sensevoice_model_device = device
```

### 效果
- 首次转写：约8秒（加载模型）
- 后续转写：约1.5秒（使用缓存）

## 7. 长音频显存不足问题

### 问题描述
较长音频（>5分钟）转写时 GPU 显存不足，报错 "CUDA out of memory. Tried to allocate 41.70 GiB"。

### 原因分析
FunASR 在 GPU 上一次性处理大音频文件，显存需求过大。

### 解决方案
检测音频时长，超过5分钟时分段处理：
```python
if duration_sec > 300:
    # 分段为5分钟一段
    for i in range(0, duration_sec, segment_duration):
        # 用 ffmpeg 提取分段
        subprocess.run([ffmpeg, '-ss', str(start), '-to', str(end), ...])
        # 逐段转写
        result = model.generate(input=segment_path, ...)
```

## 24. 支持B站短链接 (b23.tv)

### 功能描述
支持B站手机端分享的短链接格式，如 `https://b23.tv/clSQfKR`。

### 问题原因
原来的 `is_bilibili_url()` 验证函数要求 URL 必须包含 `bilibili.com`，导致短链接 `b23.tv` 格式被拒绝。但 yt-dlp 本身可以正常处理短链接的重定向。

### 实现方案

#### 24.1 添加短链接解析函数 (`speech_to_text.py`)
```python
def resolve_short_url(url, timeout=5):
    """
    解析B站短链接，获取真实长链接
    @auth: ljz
    @date: 2026-03-30
    """
    import requests
    if 'b23.tv' not in url:
        return url
    response = requests.head(url, allow_redirects=True, timeout=timeout)
    return response.url
```

#### 24.2 扩展URL验证函数
```python
def is_bilibili_url(url):
    """支持格式：长链接、短链接(b23.tv)、番剧、移动端"""
    if 'b23.tv' in url:
        return True
    return 'bilibili.com' in url and ('/video/' in url or '/bangumi/' in url)
```

#### 24.3 修改视频信息获取函数
`get_bilibili_video_info()` 和 `download_audio_from_bilibili()` 在处理 URL 前先调用 `resolve_short_url()` 解析短链接。

### 使用示例
```bash
# 现在支持短链接
python main.py https://b23.tv/clSQfKR

# 输出提示：
# [提示] 正在解析短链接: https://b23.tv/clSQfKR
# [信息] 解析成功: https://www.bilibili.com/video/BVxxx...
```

### 支持的URL格式

| URL格式 | 之前 | 现在 |
|---------|------|------|
| `https://www.bilibili.com/video/BVxxx` | ✓ | ✓ |
| `https://b23.tv/xxx` (短链接) | ✗ | ✓ |
| `https://m.bilibili.com/video/xxx` | ✗ | ✓ |
